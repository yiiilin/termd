termd 0.8.19

用户可见变化:
- 修复 iPhone Chrome PWA 中移动工作区菜单的触摸焦点时序；Files、Sessions 等菜单项不再在兼容 click 到达前被提前关闭。
- 移动 Sessions 和 Files 面板保持工具栏可操作，同时隔离被覆盖的终端工作区；键盘焦点不会落到不可见的终端控件。
- 移动工具栏的 session 标题固定在视口中心，RTT 文本变化不再推移标题；标题、延迟和终端尺寸在窄屏下保持互不遮挡。
- 移动状态栏从左侧展示 CPU、内存、磁盘和网络指标，空间不足时可横向查看，不再只剩网络指标或截断 CPU。
- 移动端常用图标按钮缩小视觉占用但保留扩展触摸区域，Files 工具栏和工作区菜单在窄屏下更紧凑。

兼容性:
- workspace、Cargo lockfile 和 Web 前端版本同步到 `0.8.19`。
- 本次只调整 Web UI、交互和相应测试，不改变 WebSocket 协议、pairing/auth、session 持久化、daemon/relay 路由或状态 schema。
- supervisor 兼容版本保持 `2026-07-12-dual-ws`；从 0.8.18 普通升级不会清空或终止已有 live session。
- GitHub Release 继续只发布 `termd`、`termctl`、`termrelay` 的 Linux amd64/arm64 六个裸二进制，不构建或推送容器镜像。

termd 0.8.4

用户可见变化:
- 修复移动端弹出软键盘后当前终端光标被推到可见区域外的问题。页面会在会话标题栏与键盘上沿之间调整终端位置，尽量让 PTY 光标保持在可见区域中部。
- 光标位于屏幕首行、中间或末行时都会按上下边界正确约束；浏览器延迟滚动、无回显输入、键盘收起和再次弹出时也会重新对齐。
- 终端的文本选择和触摸聚焦保持可用；为移动终端留出的定位空间不再触发误选中。

兼容性:
- supervisor compatibility 仍为 `2026-07-12-dual-ws`；从 0.8.3 升级到 0.8.4 不会清空现有 session、终止 supervisor、重新配对设备或重新注册 daemon。
- terminal、metadata、JSON HTTP control、认证、relay routing 和 supervisor IPC 协议均未变化。本版本只调整 Web 终端在移动软键盘下的布局与光标可见性。

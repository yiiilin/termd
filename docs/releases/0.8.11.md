termd 0.8.11

用户可见变化:
- Web 会话头像改用 DiceBear `identicon`，同一会话保持稳定且更容易在列表中区分。
- AI 状态从头像边框迁移到整个会话行背景：运行中显示蓝色呼吸灯效，完成显示绿色常亮，等待审批或操作显示黄色常亮；展开列表和折叠侧栏保持一致。
- 移除旧的头像状态外圈和状态角标，选中态、悬停态和 AI 状态可以同时辨识，不再挤占会话名称与时间信息。
- GitHub Release 流程不再构建或推送 GHCR 容器镜像，只发布 `termd`、`termctl`、`termrelay` 的 Linux amd64/arm64 六个裸二进制，缩短发版 CI 时间。

兼容性:
- workspace、Cargo lockfile 和 Web 前端版本同步到 `0.8.11`；本次 UI 状态展示不改变 daemon、relay 或 WebSocket 协议。
- supervisor 兼容版本保持 `2026-07-12-dual-ws`；从 0.8.10 普通升级到 0.8.11 不需要替换 supervisor，也不会清空已有 session。
- 使用容器部署 `termrelay` 时需从仓库 Dockerfile 自行构建镜像，或显式提供 `TERMRELAY_IMAGE`；GitHub Release 不再提供项目容器镜像。

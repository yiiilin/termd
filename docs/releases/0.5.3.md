# termd 0.5.3

## 用户可见变化

- 修复 0.5.2 发版后 CI 前端测试失败的问题；桌面浏览器窗口失焦后会正确撤销当前终端 operator 聚焦态，不再被误判成移动端软键盘过渡状态。
- 修复 release workflow 读取发版说明时仍输出占位文本的问题；新版本发版说明会优先复用 `docs/releases/<version>.md`，避免文档和 GitHub Release 内容漂移。
- 保留 0.5.2 的移动端软键盘光标居中、终端分辨率保持不变和输入焦点稳定性修复。

## 兼容性

- workspace / cargo / frontend 版本同步到 `0.5.3`。
- supervisor 兼容版本保持 `2026-06-10`；普通本地更新会保留已有 running session。
- pairing/invite 格式不变；已有配对设备和现有 session 不需要清空。
- relay 仍保持 dumb pipe，不解密、不解析业务明文。

# termd 0.5.2

## 用户可见变化

- 移动端软键盘弹出后，Web 终端会把当前光标保持在键盘上方可见区域中部，同时保持原 session 分辨率不变。
- 修复光标在终端最底部时无法靠 xterm scrollback 继续下移，导致键盘遮挡输入位置的问题；现在会用本地外层窗口裁剪完整终端网格。
- 修复移动端键盘打开期间重复光标报告可能让终端窗口高度持续膨胀、滚动位置漂移的问题。
- 修复移动端键盘态切换 session / 重建终端实例后，上一 session 的本地可视行缓存可能泄漏到新 session 的问题。
- Web 终端移动端焦点、contextmenu、beforeinput 和 visualViewport 回归覆盖增强，减少软键盘弹出后立即收起或错误上报 PTY resize 的情况。

## 兼容性

- workspace / cargo / frontend 版本同步到 `0.5.2`。
- supervisor 兼容版本保持 `2026-06-10`；普通本地更新会保留已有 running session。
- relay 仍保持 dumb pipe，不解密、不解析业务明文。
- pairing/invite 格式不变；已有配对设备和现有 session 不需要清空。

termd 0.7.2

用户可见变化:
- Web 工作台侧栏的 `Termd` 标题和守护程序管理页的 `Termd 管理` 标题现在会显示当前构建版本，例如 `v0.7.2`，便于确认 direct 或 relay 页面是否已经加载预期版本的前端资源。

兼容性:
- 本次仅增加 Web UI 版本展示，不改变双 WebSocket、JSON HTTP control、认证或 session 协议；`SUPERVISOR_VERSION` 仍为 `2026-07-12-dual-ws`，从 0.7.1 升级不需要清理现有 session、重新配对设备或重新注册 daemon。
- Web UI 嵌入在 `termd` 和 `termrelay` 二进制中；direct 与 relay 页面需要使用 0.7.2 资源，建议先升级公网 termrelay，再升级本地 termd。

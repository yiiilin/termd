termd 0.8.18

用户可见变化:
- 文件列表中的目录不再显示无意义的大小，普通文件大小保持在列表最右侧对齐。
- 文件和目录支持双击打开；桌面端可在悬浮时使用快捷操作，也可通过右键菜单完成打开、编辑、下载或删除。
- 触屏端将文件操作收敛到 `...` 菜单，桌面端隐藏重复的 `...` 入口，减少视觉噪音。
- 文件操作菜单支持方向键、Home/End、Escape、Context Menu 键和 Shift+F10，并在关闭后将焦点恢复到对应文件行。

兼容性:
- workspace、Cargo lockfile 和 Web 前端版本同步到 `0.8.18`。
- 本次只调整 Web 文件列表交互、样式和相应测试，不改变 WebSocket 协议、pairing/auth、session 持久化、daemon/relay 路由或状态 schema。
- supervisor 兼容版本保持 `2026-07-12-dual-ws`；从 0.8.17 普通升级不会清空或终止已有 live session。
- GitHub Release 继续只发布 `termd`、`termctl`、`termrelay` 的 Linux amd64/arm64 六个裸二进制，不构建或推送容器镜像。

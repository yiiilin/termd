termd 0.6.4

用户可见变化:
- 修复 Web 终端打开或 snapshot reset 期间可能短暂闪烁并失去输入焦点的问题；同一个 session reset 时会保留 xterm DOM，并在 reset 前焦点位于终端内时恢复 helper textarea 焦点。
- 修复桌面浏览器窗口切回时，helper textarea 自动恢复焦点会被误判为被动焦点并再次 blur 的问题；从后台或窗口失焦状态回来后终端输入更稳定。
- 修复移动端软键盘弹出后终端可视区域错误追随输入焦点的问题；键盘打开时终端底边会贴在快捷键栏上方，而不是把输入行推到键盘上方中间。
- 移动端键盘态仍保持原 session 行列数，不向 daemon 上报软键盘后的较小尺寸，避免触发 PTY resize 或导致键盘动画期间输入目标失稳。
- 补充移动端键盘、终端 reset 焦点恢复和布局回归测试，覆盖重复贴底、无底部留白和全量前端回归。

兼容性:
- 这是 0.6.3 的 Web UI 稳定性修复；不改变 pairing 流程、不改变 packet protocol、不改变 supervisor 兼容版本、不要求清空 session。
- 建议 daemon、Web UI 和 termrelay 同步升级到 0.6.4，确保公网 relay 页面也加载修复后的前端资源。

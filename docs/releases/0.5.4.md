# termd 0.5.4

## 用户可见变化

- 修复 0.5.3 release QA 中真实 relay 主干断开重连回归的测试漂移；重连验收不再依赖 session 列表刷新时序，而是直接验证当前已打开终端能否恢复输入和输出。
- 补强 Web 终端的移动端 `contextmenu` / `beforeinput` 回归覆盖，避免迟到的长按菜单事件误清 helper textarea 输入通道，减少软键盘弹出后立刻失焦的问题。
- 补强 xterm 调试镜像与选择测试基线：在 snapshot redraw / resize stabilizing 期间额外暴露实际 `rows/cols`，修复下半区拖拽选区和 CI 高 DPI 环境下的误判。
- release QA 改为单 worker 跑前端 Playwright，降低真实 termd/termrelay/Vite 同机并发带来的端口和网络抖动，使发版验证结果更稳定可复现。

## 兼容性

- workspace / cargo / frontend 版本同步到 `0.5.4`。
- supervisor 兼容版本保持 `2026-06-10`；普通本地更新会保留已有 running session。
- pairing/invite 格式不变；已有配对设备和现有 session 不需要清空。
- relay 仍保持 dumb pipe，不解密、不解析业务明文。

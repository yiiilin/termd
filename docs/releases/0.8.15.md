termd 0.8.15

用户可见变化:
- 修复 Git Changes 列表中只有右侧小图标可以打开 diff 的问题；现在点击文件名或变更行空白区域都能查看文件差异。
- Git 变更行支持使用 Enter 或 Space 打开 diff，并保持行内操作按钮只执行自身操作，不会重复触发打开动作。

兼容性:
- workspace、Cargo lockfile 和 Web 前端版本同步到 `0.8.15`；本次只调整内嵌 Web UI，不改变 daemon、relay 或 WebSocket 协议。
- supervisor 兼容版本保持 `2026-07-12-dual-ws`；从 0.8.14 普通升级不需要替换 supervisor，也不会清空已有 session。
- GitHub Release 继续只发布 `termd`、`termctl`、`termrelay` 的 Linux amd64/arm64 六个裸二进制，不构建或推送容器镜像。
